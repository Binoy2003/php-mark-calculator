# php-mark-calculator
A simple PHP web application to calculate the best two class test (CT) marks, total marks, and pass/fail status based on student scores.
# PHP Mark Calculator

A simple PHP project that calculates a student's total marks.

## Features

- Takes input for 3 Class Tests (CT1, CT2, CT3)
- Selects the best two CT marks
- Calculates total marks
- Displays Pass/Fail status

## Technologies Used

- PHP
- HTML
- XAMPP

## Formula

Total Marks = Best Two CT + Midterm + Final
